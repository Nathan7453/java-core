# java-core

> 实践环境说明如下:

```
* Java11(11.0.1)
```

```
com.nathan.core
    ├─ array(数组)
    ├─ oop(面向对象)
    ├─ string(String)
    ├─ exception(Exception)
    ├─ reflect(Reflect)
    ├─ collection(Collection)
    ├─ generic(泛型)
    ├─ io(I/O)
    │    ├─ (字节流)
    │    └─ (字符流)
    ├─ net(网络编程)
    └─ juc(Java)
        ├─ thread
        └─ 
```
